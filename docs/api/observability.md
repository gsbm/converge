# converge.observability

Logging, tracing, metrics, and replay. **Logging**: JsonFormatter for structured logs, configure_logging, get_logger, log_struct. **Tracing**: trace context manager and get_current_trace_id for span tracking. **MetricsCollector**: counters and gauges with snapshot(). **ReplayLog**: record messages and export/load event logs for replay and inspection.
